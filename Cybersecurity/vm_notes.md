# Notes

## Snapshots

### Create snapshot

```sh
savevm <snapshot_name>
```

### Load snapshot

```sh
loadvm <snapshot_name>
```

### List snapshots

```sh
info snapshots
```

### Delete snapshots

```sh
delvm <snapshot_name>
```
